# CSE - Connect

## Overview
CSEConnect is a B.Tech CSE-focused learning management system with two user roles: Student and Faculty. Students can browse and purchase courses, watch video lectures, take quizzes, and submit assignments. Faculty can create and manage courses, add content, and monitor student enrollments and progress.

## Features

### For Students
**Browse Courses**:- View all published courses with title, description, thumbnail, price, and lecture count.

**Course Purchase**:- Buy a course with one click via a dummy payment system. Duplicate purchases are blocked.

**Free Enrollment**:- Free courses can be enrolled in without any payment step.

**Dashboard - Live stats**:- enrolled courses, completed lectures, quizzes done, assignments submitted.

**Progress Tracking**:- Per-course progress bar updated in real time as you complete lectures, quizzes, and assignments.

**Lecture Player**:- Watch video lectures embedded directly in the learn page.

**Mark as Watched**:- Manually mark a lecture as watched after viewing.

**Quizzes**:- Multiple-choice quizzes per lecture with instant scoring on submission.

**Assignments**:- Text-based assignment submissions per lecture.

**Lecture Status Indicators**:- Visual indicators (complete / not started) in the sidebar.

### For Faculty
**Create Courses**:- Add a new course with title, description, thumbnail, price, category, and free/paid toggle.

**Manage Lectures**:- Add, edit, reorder, and delete lectures within a course.

**Add Quizzes**:- Build multiple-choice quizzes with a dynamic question builder (multiple questions, 4 options each, correct answer selector).

**Add Assignments**:- Create text-based assignments with title, description, and optional due date.

**View Enrollments**:- See all enrolled students per course with their name, email, enrollment date, amount paid, and progress percentage.
