Operating Systems

Understand the core concepts of operating systems including process management, memory management, file systems, and concurrency.

16 Weeks 4 Credits Dr. Rajesh Kumar

Syllabus

  1. Introduction to Operating Systems
  2. Process Management
  3. CPU Scheduling Algorithms
  4. Deadlocks
  5. Memory Management
  6. Virtual Memory
  7. File Systems
  8. Input / Output Systems

Learning Outcomes

  • Understand how modern operating systems work internally
  • Analyze scheduling and memory management techniques
  • Identify and resolve deadlocks and synchronization issues
  • Apply OS concepts to system-level problem solving
