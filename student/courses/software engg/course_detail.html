Software Engineering

Learn systematic approaches to software development including requirements analysis, design, implementation, testing, and maintenance.

12 Weeks 3 Credits Dr. Priya Nair

Syllabus

  1. Introduction to Software Engineering
  2. Software Development Life Cycle (SDLC)
  3. Requirements Engineering
  4. Software Design Models
  5. Agile & Scrum Methodologies
  6. Software Testing Techniques
  7. Software Maintenance
  8. Project Management

Learning Outcomes

  • Understand structured software development processes
  • Apply SDLC and agile methodologies effectively
  • Design and test reliable software systems
  • Manage software projects and documentation
