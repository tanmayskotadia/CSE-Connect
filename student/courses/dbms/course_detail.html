Database Management Systems

Learn the concepts of database design, relational models, SQL, normalization, transaction management, and data integrity.

12 Weeks 3 Credits Prof. Anita Sharma

Syllabus

  1. Introduction to Databases
  2. Entity–Relationship Model
  3. Relational Model & Constraints
  4. SQL Queries and Joins
  5. Normalization Techniques
  6. Transaction Management
  7. Concurrency Control
  8. Database Security

Learning Outcomes

  • Design efficient database schemas using ER models
  • Write complex SQL queries for data retrieval
  • Apply normalization to reduce data redundancy
  • Understand transactions, concurrency, and security
