Machine Learning

Learn how machines learn from data using supervised and unsupervised techniques, neural networks, and model evaluation methods.

16 Weeks 4 Credits Dr. Suresh Iyer

Syllabus

  1. Introduction to Machine Learning
  2. Data Preprocessing & Feature Engineering
  3. Supervised Learning Algorithms
  4. Unsupervised Learning Algorithms
  5. Model Evaluation & Validation
  6. Neural Networks & Deep Learning Basics
  7. Ensemble Learning Techniques
  8. Ethics & Applications of Machine Learning

Learning Outcomes

  • Understand core concepts of machine learning
  • Build and evaluate ML models using real datasets
  • Select appropriate algorithms for different problems
  • Apply ML techniques to real-world applications
