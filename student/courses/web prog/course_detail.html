Web Programming

Learn to design and develop modern, responsive web applications using HTML, CSS, JavaScript, and an introduction to front-end frameworks.

16 Weeks 4 Credits Dr. Priya Nair

Syllabus

  1. Introduction to Web Technologies
  2. HTML5 Fundamentals
  3. CSS3 & Responsive Design
  4. JavaScript Basics
  5. DOM Manipulation
  6. Client-Side Form Validation
  7. Introduction to Front-End Frameworks
  8. Web Application Deployment

Learning Outcomes

  • Build responsive and interactive web pages
  • Use HTML, CSS, and JavaScript effectively
  • Develop client-side logic and validations
  • Deploy basic web applications
