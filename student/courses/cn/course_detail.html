Computer Networks

Understand the fundamentals of data communication, network architectures, protocols, and the functioning of modern computer networks.

12 Weeks 3 Credits Dr. Meena Iyer

Syllabus

  1. Introduction to Computer Networks
  2. Network Models (OSI & TCP/IP)
  3. Physical and Data Link Layers
  4. Network Layer & Routing Algorithms
  5. Transport Layer Protocols (TCP & UDP)
  6. Application Layer Protocols
  7. Network Security Basics
  8. Wireless and Mobile Networks

Learning Outcomes

  • Understand the working principles of computer networks
  • Explain network models and protocol layers
  • Analyze routing and transport mechanisms
  • Apply networking concepts to real-world scenarios
